# Code signing policy

Free code signing provided by SignPath.io, certificate by SignPath Foundation.

## Team roles

- Authors: Justin Barry
- Reviewers: Justin Barry
- Approvers: Justin Barry

## Privacy policy

This program will not transfer any information to other networked systems unless specifically requested by the user or the person installing or operating it.

## Release process

- All Windows release binaries are built from this repository.
- Each release is manually approved before signing.
- Signed binaries are published through GitHub Releases for this project.
