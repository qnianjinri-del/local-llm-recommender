# Local LLM Recommender
一键识别电脑硬件，推荐适合本机的开源大模型，并通过 Ollama 一键部署。

## 下载方式
请前往本仓库的 **Releases** 页面，下载最新版本：
- `LocalLLMRecommender.exe`

## 适合谁使用
这个工具面向：
- 想在本地运行开源大模型的普通用户
- 不熟悉 Python、命令行、模型部署的 AI 小白
- 想快速判断自己电脑适合跑什么模型的用户

## 当前已实现功能
- 自动识别本机硬件信息
- 联网获取近期更新的模型家族
- 根据硬件配置推荐适合的开源模型
- 通过 Ollama 一键下载并部署模型
- 在桌面界面中直接测试模型对话
- 支持检测和引导启动 Ollama

## 使用前准备
使用本工具前，请先安装 **Ollama**。
如果尚未安装，打开程序后可点击：
- **检测 / 启动 Ollama**
程序会尝试自动启动 Ollama，或者引导你打开下载页面。

## 使用步骤
1. 前往 **Releases** 页面下载 `LocalLLMRecommender.exe`
2. 双击运行程序
3. 如未启动 Ollama，可先点击 **检测 / 启动 Ollama**
4. 点击 **开始扫描并推荐**
5. 查看当前电脑适合的模型
6. 点击推荐模型卡片中的 **设为当前模型**
7. 点击 **一键部署当前模型**
8. 在测试问题中输入内容
9. 点击 **发送测试消息** 查看模型回复
